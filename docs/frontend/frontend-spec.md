# Frontend Specification

## Overview

The frontend provides the user interface for the application.

It is implemented using Next.js.

---

## Responsibilities

Provide search interface

Send API requests

Receive streaming results

Display product comparisons

---

## UI Sections

Navbar

Search section

Results section

Product comparison cards

## Design Assets

Design inspiration images are stored in:

frontend/design/

These images serve as visual references for the UI style of the application.

The design direction includes:

- pastel color themes
- rounded card components
- minimal and clean layout
- centered search interface

Developers should use these assets as inspiration when implementing the UI.
